# 🎮 실시간 테트리스 배틀 (교실용)

반 전체가 각자 기기(노트북·태블릿·휴대폰)로 접속해서 **동시에 실시간 테트리스 대결**을 하는 프로그램입니다.
교사 컴퓨터 한 대를 서버로 쓰고, 학생들은 같은 와이파이로 접속만 하면 됩니다.

- 2줄 이상 지우면 다른 학생에게 **방해 줄**이 날아갑니다.
- 마지막까지 살아남거나 최고 점수면 **우승**!
- 교사 화면(상황판)에서 모든 학생의 보드와 실시간 순위를 한눈에 볼 수 있습니다.
- **외부 라이브러리가 전혀 없습니다.** Node.js만 있으면 바로 실행됩니다.

---

## 1. 준비 (교사 컴퓨터, 1회만)

1. **Node.js 설치** — https://nodejs.org 에서 LTS 버전 설치 (이미 있으면 생략)
   - 설치 확인: 터미널/명령 프롬프트에서 `node -v` 입력 → 버전이 보이면 OK

2. 이 폴더(`tetris-battle`)를 컴퓨터에 풀어둡니다.

## 2. 서버 실행

폴더 안에서 터미널을 열고:

```
node server.js
```

실행하면 이런 안내가 나옵니다:

```
상황판(교사 화면) 열기:
   http://localhost:3000/?host=1

학생 접속 주소 (같은 와이파이):
   http://192.168.0.15:3000      ← 이 주소를 학생들에게!
```

> 포트를 바꾸고 싶으면: `PORT=4000 node server.js`

## 3. 교실에서 진행

1. **교사**: 프로젝터 연결한 컴퓨터에서 `http://localhost:3000/?host=1` 을 엽니다. → 상황판이 뜹니다.
2. **학생**: 자기 기기 브라우저에서 위에 나온 **학생 접속 주소**(예: `http://192.168.0.15:3000`)를 입력 → 이름 입력 후 입장.
3. 학생들이 다 들어오면 상황판에서 **▶ 시작** 버튼 클릭 → 3·2·1 카운트다운 후 전원 동시 시작!
4. 끝나면 **🔄 리셋** → 다시 **▶ 시작** 으로 새 라운드.

## ⚠️ 학생이 접속이 안 될 때

대부분 **학교 와이파이 문제**입니다. 아래를 확인하세요.

1. **교사 컴퓨터와 학생 기기가 같은 와이파이**에 있어야 합니다. (교무용/학생용 망이 분리돼 있으면 안 됩니다.)
2. **방화벽**이 막을 수 있습니다 → 교사 컴퓨터에서 Node.js의 네트워크 접근을 "허용"으로 설정하세요.
   - 윈도우: 처음 실행 시 뜨는 방화벽 창에서 **액세스 허용** 클릭.
3. 일부 학교망은 기기끼리 통신을 차단(클라이언트 격리)합니다. 이 경우 같은 망인데도 접속이 안 됩니다.
   - 임시 해결: 교사 컴퓨터로 **휴대폰 핫스팟**을 켜고, 교사 PC와 학생들이 모두 그 핫스팟에 접속.
   - 또는 클라우드 배포(아래) 사용.

## 직접 고쳐보기 (바이브코딩 팁)

- **게임 속도**: `public/index.html`의 `dropInterval` 함수 숫자를 바꾸면 난이도 조절.
- **방해 줄 양**: `clearLines` 안의 `[0,0,1,2,4]` 배열이 "지운 줄 → 보낼 방해 줄" 표입니다.
- **점수**: 같은 함수의 `[0,100,300,500,800]`.
- **색깔**: `COLOR` 배열.
- 서버 로직은 `server.js`, 화면·게임은 `public/index.html` 한 파일에 모두 들어 있습니다.
